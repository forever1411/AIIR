# Portfolio Records

本目录保存用户明确授权持久化的投资组合信息，使 AIIR 能在重启会话或更换模型后恢复最新已知持仓，而不要求用户重复上传未变化的数据。

## Authorization and visibility

本仓库当前公开可访问。用户已明确知晓这一点，并授权在本目录中保存：

- 股票名称和证券代码；
- 持有与可用数量；
- 平均成本和快照价格；
- 市值、参考盈亏、收益率和账户仓位；
- 由以上数据计算或合理推断的组合汇总。

该授权不包括券商账号、登录凭据、认证密钥、身份号码、银行卡信息、税务资料、原始对账单或原始账户截图。这些内容不得写入公开仓库。

## Files

- [`CURRENT.md`](CURRENT.md)：最新已知持仓。后续会话涉及组合时优先读取。
- [`snapshots/`](snapshots/)：按日期保存的重要历史快照，用于复盘持仓和组合变化。

## Update rules

- 每条记录必须标明数据日期、来源、币种，以及直接读取、计算和推断数据的区别。
- 用户报告成交、数量或成本变化时，更新 `CURRENT.md`；组合发生实质变化或需要建立复盘节点时，新增日期快照。
- 普通市场价格波动不要求每天写入仓库。分析实时情况时，应另行检索最新价格和公告，不把旧快照价格当成当前行情。
- 未收到交易变化时，数量和成本保持为最新已知值；不得从价格变化推断用户已经交易。
- 历史快照不静默改写。若发现录入错误，应显式记录纠正及原因。
- 下次启动后，不要求用户重新上传相同持仓；只确认自最新快照以来发生的变化或无法可靠判断的部分。
