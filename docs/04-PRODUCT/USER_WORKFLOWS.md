# AIIR 用户工作流总览

## 1. 学习工作流

```text
我不懂一个概念
→ 系统判断与当前目标的关系
→ 用简单语言解释
→ 给一个例子
→ 检查理解
→ 推荐下一步
```

适用于：股票、债券、基金、财报、估值、风险、分散、再平衡等。

## 2. 引导研究工作流

```text
模糊问题
→ 明确研究目的
→ 补全未知问题
→ 找可靠来源
→ 提取事实
→ 解释含义
→ 提出反方证据
→ 列出未知
→ 决定下一步研究
```

详见 `docs/09-WORKFLOWS/GUIDED_RESEARCH_WORKFLOW.md`。

## 3. 资产配置讨论工作流

```text
目标和期限
→ 现金与负债
→ 风险承受能力
→ 当前资产暴露
→ 一般配置原则
→ 多个候选方案
→ 压力情景
→ 待核实问题
→ 用户决定
```

不得从“我该怎么配置”直接跳到精确比例。

## 4. 决策准备工作流

```text
拟采取行动
→ 动机检查
→ 个人约束
→ 事实和来源
→ 支持理由
→ 反方理由
→ 仓位和替代方案
→ 失效条件
→ 冷静期 / 下一步
→ 用户决定
```

详见 `docs/09-WORKFLOWS/DECISION_WORKFLOW.md`。

## 5. 复盘工作流

```text
恢复当时记录
→ 区分当时可知与后来事实
→ 检查过程质量
→ 区分判断与运气
→ 识别偏误
→ 更新规则候选
→ 保留原记录
```

详见 `docs/09-WORKFLOWS/REVIEW_WORKFLOW.md`。

## 6. Friction 工作流

```text
出现不便或错误
→ 记录具体场景
→ 使用低成本办法修复
→ 观察是否重复
→ 评估影响
→ 达到条件后进入软件候选池
```

任何开发任务都必须能回到这条链路。
