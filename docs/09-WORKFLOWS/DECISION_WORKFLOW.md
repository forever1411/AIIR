# 投资决策准备工作流

## 1. 原则

AIIR 帮助用户准备决定，不替用户决定。

任何“我该不该买、卖、加仓、减仓”的问题，都先通过本流程。

## 2. 决策输入

- 拟采取的行动；
- 资产或策略；
- 原始动机；
- 计划金额或仓位；
- 资金用途和时间期限；
- 当前组合关系；
- 已有研究；
- 用户最担心的问题。

## 3. 检查步骤

### 3.1 动机检查

- 是因为基本面、配置需要，还是因为价格、热点或他人观点？
- 是否存在 FOMO、恐慌或回本心理？
- 如果今天没有持仓，是否仍会做同样决定？

### 3.2 财务边界

- 应急储备是否充足？
- 是否存在更优先的高成本债务？
- 资金何时需要？
- 最坏损失是否可承受？

### 3.3 研究边界

- 是否理解资产如何产生回报？
- 核心事实来自哪里？
- 哪些是假设？
- 最强反方证据是什么？
- 哪些信息仍缺失？

### 3.4 组合边界

- 行动后单一资产占比；
- 行业、地区、币种和风格集中度；
- 与已有持仓是否高度重复；
- 是否存在更分散、更简单或更低成本替代方案。

### 3.5 行动设计

讨论多个选项，例如：

- 不行动；
- 继续研究；
- 等待特定信息；
- 小额试验；
- 分批；
- 调整其他暴露；
- 寻求专业意见。

AIIR 不默认将“有观点”转化为“必须交易”。

### 3.6 失效条件

用户必须能够说明：

- 什么事实说明判断错误；
- 什么风险超出承受范围；
- 何时复盘；
- 哪些条件下不因短期价格变化行动。

## 4. 决策记录

用户确认后记录：

```text
日期
决定
当时的个人和组合背景
主要理由
使用的来源
反方观点
未知
替代方案
计划仓位或限制
失效条件
复盘日期
```

## 5. 输出结论等级

- **Not Ready**：缺少关键个人事实或研究；
- **Research Next**：问题明确但证据不足；
- **Decision Ready**：信息和风险检查基本完成；
- **Professional Review Recommended**：涉及复杂税务、法律、杠杆或重大生活资金。

等级不等于买卖评级。
