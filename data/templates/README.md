# AIIR 私有数据模板

这些文件是公开的虚构模板。

使用时：

```bash
mkdir -p workspace/private
cp data/templates/investment_profile.example.md workspace/private/investment_profile.md
cp data/templates/portfolio.example.csv workspace/private/portfolio.csv
cp data/templates/watchlist.example.csv workspace/private/watchlist.csv
cp data/templates/investment_rules.example.md workspace/private/investment_rules.md
cp data/templates/decision_journal.example.md workspace/private/decision_journal.md
cp data/templates/friction_log.example.md workspace/private/friction_log.md
```

然后填写真实数据。`workspace/private/` 已被 `.gitignore` 排除。

不要将真实持仓、成本、账户信息或个人财务背景提交 Git。
