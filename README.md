# AIIR

> **AI Investment Research**
>
> AIIR 是一个 AI 驱动的专业投资研究平台，致力于帮助普通投资者建立专业投资机构级别的研究能力，而不是替代用户做投资决策。

---

# 项目简介

AIIR（AI Investment Research）诞生于一个简单的问题：

> **如果我身边有一位工作了几十年的专业投资研究员，每天帮我收集信息、分析市场、研究公司、讨论观点，我的投资决策会不会更好？**

AIIR 希望借助 AI，将这种能力变成现实。

它不是一个炒股软件，也不是一个自动交易机器人。

它更像一位专业、理性、长期陪伴的投资研究伙伴。

---

# 项目目标

AIIR 的目标不是预测股价，而是帮助用户：

- 建立专业投资研究能力
- 提升投资认知
- 理解市场运行逻辑
- 构建属于自己的投资体系
- 实现长期、稳健的财富增长

---

# AIIR 不是什么

AIIR 不会：

- 自动买卖股票
- 替用户做投资决策
- 承诺收益
- 预测未来市场
- 输出没有依据的结论

最终投资决策始终由用户完成。

---

# AIIR 能做什么

未来 AIIR 将逐步具备以下能力：

### 投资研究

- 股票研究
- 公司基本面分析
- 财务分析
- 行业分析
- 宏观经济分析

### 信息收集

- 全球财经新闻
- 上市公司公告
- 财报
- 政策信息
- 市场热点
- 社交媒体观点（按需）

### AI 分析

- 多源信息整合
- 事件影响分析
- 风险识别
- 投资逻辑推理
- 多观点讨论

### 知识管理

- 长期知识库
- 投资案例积累
- 用户研究历史
- 决策复盘

### 智能报告

自动生成：

- 个股研究报告
- 每日晨报
- 市场周报
- 行业专题研究
- 风险分析报告

# 开发原则

AIIR 坚持以下原则：

- Design First（先设计）
- Documentation First（先文档）
- Evidence First（证据优先）
- Human Decision（用户最终决策）
- Git is Source of Truth（Git 是唯一事实来源）

详细规范请参阅 `CONSTITUTION.md`。
