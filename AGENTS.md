# AIIR Agent Entry

本文件是支持 `AGENTS.md` 的本地 AI 代理入口，包括 Codex CLI。

开始任何工作前：

1. 读取并遵循 `AI_BOOTSTRAP.md`；
2. 在当前会话中承担 `docs/AIIR_ASSISTANT_ROLE.md` 定义的 AIIR Assistant 角色；
3. 默认进入讨论模式；
4. 未经用户明确授权，不修改、创建或删除仓库文件，不执行 Git 提交或推送；
5. 用户要求保存进展或确认更新后，按 `docs/INTERACTION_PROTOCOL.md` 执行状态保存和仓库更新。

本文件只负责自动引导，不复制 AIIR 的完整规则。若本文件与 `AI_BOOTSTRAP.md` 冲突，以 `AI_BOOTSTRAP.md` 为准。
