# AIIR Current State

- **Last updated:** 2026-07-23
- **Status:** Minimal operating system kernel being established

## Current direction

AIIR 不再开发独立本地程序。GitHub 仓库作为长期记忆和正式事实来源，ChatGPT 等成熟大模型作为研究、解释、推理、质疑和交互引擎。

当前目标是先让以下闭环稳定运行：

```text
启动恢复
→ 真实交流与工作
→ 状态保存检查
→ 用户确认
→ 仓库更新
→ 下次恢复
```

## Current user need

用户目前投资经验有限，不知道应从哪里开始，也不应被要求先学会提出专业投资问题。AIIR 需要主动告诉用户：

- 现在应该了解什么；
- 应该问什么；
- 应该看什么资料；
- 哪些地方尚未理解或证据不足；
- 做决定前还缺什么。

## Current repository task

将仓库重建为最小但完整的 AIIR 系统内核，包括：

- 项目定义和 Vision；
- 大模型角色；
- 基本原则；
- 交互、启动和保存协议；
- 当前状态；
- 长期记忆；
- 历史日志。

不保留软件开发、Prompt 库、模板库或没有当前用途的目录。

## Known gaps

- 尚未开始录入具体个人投资背景、财务目标或持仓；
- 尚未运行重建后的第一次真实投资学习或研究会话；
- 状态保存协议尚需通过实际会话验证和微调。

## Next step

应用并提交本次仓库重建 patch。下一次启动后，直接运行第一次真实 AIIR 会话；只在实际交流中自然产生相关信息时，再逐步丰富 `MEMORY.md`、`CURRENT_STATE.md` 和 `JOURNAL.md`。
