
## Current user need

用户目前投资经验有限，需要 AIIR 主动帮助：

- 找到当前最值得解决的问题；
- 确定合理的学习或研究起点；
- 生成用户尚不知道应该提出的问题；
- 找到并解释相关资料；
- 识别风险、未知和决策前缺口；
- 降低维护长期上下文的负担。
