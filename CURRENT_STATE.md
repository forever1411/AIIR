# AIIR Current State

- **Last updated:** 2026-07-23
- **Status:** Ready for operation

## Current objective

让 AIIR 的基本闭环通过真实会话开始运行：

```text
恢复上下文
→ AIIR 助手主动协助用户解决实际问题
→ 识别新的有效信息
→ 用户确认
→ 更新仓库
→ 下次继续
```

## Current user need

用户目前投资经验有限，需要 AIIR 助手主动帮助：

- 找到当前最值得解决的问题；
- 确定合理的学习或研究起点；
- 生成用户尚不知道应该提出的问题；
- 找到并解释相关资料；
- 识别风险、未知和决策前缺口；
- 降低维护长期上下文的负担。

## Available context

当前仓库已经定义：

- AIIR 的目标与运行方式；
- 工具中立的 AIIR Assistant 角色和责任边界；
- 基本原则；
- 会话启动、讨论、保存和更新协议；
- 已确认的用户背景与协作偏好。

AIIR 可以通过远程仓库方式在网页模型中运行，也可以通过本地仓库方式由 Codex 等 AI 代理运行。

## Information to build through use

以下信息尚未形成正式记录，应只在实际问题需要时逐步确认：

- 个人财务目标与时间期限；
- 收入、支出、应急资金和负债概况；
- 风险承受能力与流动性需求；
- 当前资产、持仓和观察对象；
- 已有投资决定及其理由；
- 需要优先学习或研究的主题。

## Next step

从用户当前最关心的个人理财或投资问题开始第一次正式会话。AIIR 助手应利用已有上下文主动引导；只有结论确实依赖缺失信息时，才提出少量高价值问题。
