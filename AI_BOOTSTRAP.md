# AIIR Bootstrap

如果你是首次加入 AIIR 项目，请按以下顺序阅读：

1. README.md
2. PROJECT_CONTEXT.md
3. PROJECT_STATUS.md
4. docs/00-VISION/00-VISION.md
5. docs/01-GOVERNANCE/CONSTITUTION.md
6. docs/03-ARCHITECTURE/

阅读完成后：

- 理解项目定位
- 确认当前 Sprint
- 再开始讨论或开发

重要原则：

Git 是唯一事实来源（Single Source of Truth）。
聊天记录仅用于讨论，不作为正式项目依据。
